> Minimal, developer-focused Hugo theme for portfolio, blog, and personal branding.

An open-source Hugo theme designed for developers and DevOps engineers who want a clean, fast, and maintainable personal website. The theme focuses on readability, simplicity, and a strong developer-centric aesthetic.

It provides structured layouts for blog posts, project portfolios, and resume-style content, enabling users to present technical work in a clear and professional way. The design follows a typography-first approach with built-in dark mode support.

The theme is built with simplicity in mind, avoiding unnecessary dependencies while remaining flexible and easy to customize.

## Key Features

- Clean and minimal design optimized for readability
- Dark-mode-first styling
- Dedicated layouts for blog, projects, and resume
- Fast build times and lightweight structure
- Easy customization without complex tooling

## Technologies & Methods

- Hugo (static site generator)
- HTML / CSS (custom theming)
- Responsive design principles
- Content-driven architecture

## Impact

- Provides a reusable open-source theme for developer portfolios
- Enables fast and maintainable personal website setups
- Serves as the foundation for my own portfolio and blog
