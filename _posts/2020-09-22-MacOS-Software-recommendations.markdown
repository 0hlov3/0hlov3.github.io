Today i want to show you, wich MacOS Apps make my life easier, macOS is stable and efficient. But Apple doesn't listen to the Users, so they don't add simple features to make the life of an Apple User easier. Thankfully there are some developer who code Apps, that can help you to get more efficient.

Here is a list of my most used Apps, that helps me a lot.

## Rectangle
#### "Move and resize windows in macOS using keyboard shortcuts or snap areas" 
Rectangle is a cool tool that can help you to get more out of your window manager. With Rectangle you can set keyboard shortcuts that position the windows accordingly, you can set snap areas, that means that if you drag a window to a certain position, it will be positioned there as desired.

[Github](https://github.com/rxhanson/Rectangle){:target="_blank"} \
[Rectangle](https://rectangleapp.com/){:target="_blank"} \

## CopyClip
### Clipboard Management Done Right
CopyClip is the simplest and most efficient clipboard manager for your Mac. Running discreetly from your menu bar, this app stores all that you have copied or cut in the past, allowing you to quickly find that snippet of text you've been looking for.
(copied from the App Page, cause i have nothing else to say about CopyClip)

The good side is that you can select apps that have to be excluded and not store clips, i.e. you can exclude your password manager, so the passwords are not stored in CopyClip.

[CopyClip App Store](https://apps.apple.com/us/app/copyclip-clipboard-history/id595191960){:target="_blank"}

## Amphetamine
Amphetamine, is the most awesome keep-awake app ever created for macOS. Amphetamine can keep your Mac, and optionally its display(s), awake through a super simple on/off switch, or automatically through easy-to-configure Triggers. Amphetamine is extremely powerful and includes advanced features for those who need them, yet remains intuitive and easy-to-use for those who don't need all of the bells and whistles.

## CleanMyMac
#### CleanMyMac X is all-in-one package to awesomize your Mac. It cleans megatons of junk and makes your computer run faster. Just like it did on day one.
CleanMyMac is an awesome tool that lets you clean up your Mac, delete software and make settings. I use CleanMyMac mainly to clean up my Mac in between and to see where large unused files are stored. Furthermore you have the possibility to delete software, whereby CleanMyMac looks which files belong to this software and removes them as well. With CleanMyMac you can also manage login items and launch agents. I simply love this tool.

## iTerm2
#### iTerm2 is a terminal emulator for macOS that does amazing things
iTerm2 is the replacement for the native terminal that comes with MacOS and is the successor of iTerm. iTerm2 is optimized for MacOS 10.12 or newer and is a really fast and efficient terminal emulator. iTerm2 brings the terminal into the modern age with features you never knew you always wanted. For example, you can view the battery status, RAM usage, CPU usage and many other useful things. iTerm2 supports split-screen, tabs and one of the best features for me is broadcast input.

There are a lot of more settings, functions and possibilities to configure iTerm2, but it would be worth a blog entry to go into more detail here.

[iTerm2](https://iterm2.com) \
![CleanMyMacX](/assets/macossoftwarerecomm/iterm2.png){:height="auto" width="100%"}

## Sublime Text
#### A sophisticated text editor for code, markup and prose

Sublime Text is a beautiful, feature-rich text editor for editing code with a focus on usability. Its best features include split editing as well as quick shortcuts and search functions.
Unlike some other text editors, Sublime Text is unfortunately not free and at $80 for a license, Sublime Text is not cheap, but it can be worth it. Because the text editor is really intuitive and has some very useful features. For example, I love that you can edit several lines at once.

[Sublime Text](https://www.sublimetext.com){:target="_blank"} \

## Affinity Photo
### Picture the future.
Affinity Photo has become the first choice for photography and creative professionals around the world, who love its speed, power and precision. Born to work hand-in-hand with the latest powerful computer technology, it’s the only fully-loaded photo editor integrated across macOS, Windows and iOS.

For me, Affinity Photo is one of the best photo editing programs for Mac users who don't want to pay subscription-based prices and still be able to get top performance. 
You get all these features and more for a one-time fee. The user interface of Affinity Photo is very intuitive and I got along with the photo editor right away, once you get familiar with the software you will find that Affinity Photo is exactly the app you were looking for. The numerous tools, effects and filters make it possible to get really creative. One of the coolest features Affinity Photo has to offer is a Before and After view to compare the original photo with its edited version.
